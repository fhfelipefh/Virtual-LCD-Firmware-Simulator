# lcd-renderer

`lcd-renderer` displays a virtual LCD framebuffer in a desktop window and composites it into SVG bezels.

It is intended to be used together with `lcd-core`:

```toml
[dependencies]
lcd-core = "0.1.0"
lcd-renderer = "0.1.0"
```

Repository: <https://github.com/fhfelipefh/Virtual-LCD-Firmware-Simulator>
